github pages绑定域名

·在本地hexo目录的source文件夹中建立CNAME文件，内容为要绑定的域名，如: myurl.com。hexo发布时会将CNAME发布到blog的根目录。
·在万网设置域名解析，类型选择CNAME，主机记录空白不填，记录值写github pages的url。保存后等待若干（大概是10+）分钟后生效。
